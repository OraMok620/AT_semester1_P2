# AT_semester1_P2
Emotion responsive app

This is an emotion responsive mp3 player
made with html, js and css
as this is a progressive web application (Pwa), and the project requirement is to submit an app
Select index.html, Go Live, and download through the download icon 
(look How_To_Install.jpg)

Steps for using emotion responsive music player:
1. click folder icon in top right
2. add music to each folder
3. emotion detected and switch to the folder that match your mood
4. press play to listen to music you added
